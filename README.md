## 🎯 关于本项目

计算机基础是每个程序员的**核心竞争力**，也是考研 408 统考的四座大山：

> **数据结构 → 计算机组成原理 → 操作系统 → 计算机网络**

这个仓库用 **代码实现** 来打通每门课的理论与实践。不只看书，而是把书上的伪代码变成能跑、能调试、能对比的 C++ 程序。

目前已有**数据结构**的完整代码实现，其余三门课逐步更新中。

### 排序算法

| 文件 | 算法 | 时间复杂度 | 稳定性 |
|------|------|-----------|-------|
| `11基础排序（冒泡排序）.cpp` | 冒泡排序 | O(n²) | ✅ 稳定 |
| `11基础排序（选择排序）.cpp` | 选择排序 | O(n²) | ❌ 不稳定 |
| `11基础排序（插入排序）.cpp` | 插入排序 | O(n²) | ✅ 稳定 |
| `11基础排序（希尔排序）.cpp` | 希尔排序 | O(n log n)~O(n²) | ❌ 不稳定 |
| `12快速排列.cpp` | 快速排序 | O(n log n) | ❌ 不稳定 |
| `13归并排序.cpp` | 归并排序 | O(n log n) | ✅ 稳定 |
| `15堆排序.cpp` | 堆排序 | O(n log n) | ❌ 不稳定 |
| `17基数排序.cpp` | 基数排序 | O(d·n) | ✅ 稳定 |

排序算法对比：
```
时间复杂度竖轴：
O(n²)      冒泡 · 选择 · 插入
O(n log n) 希尔 · 快排 · 归并 · 堆排序
O(d·n)     基数排序（d 为位数）

稳定性横轴：
稳定   → 冒泡 · 插入 · 归并 · 基数
不稳定 → 选择 · 希尔 · 快排 · 堆排序
```

## 🚀 如何开始

### 环境要求

- C++17 兼容的编译器（g++ / clang++ / MSVC）
- 支持 C++11 及以上标准

### 编译运行

```bash
# 克隆仓库
git clone https://github.com/hyyhyy07/cs-408-foundation.git
cd cs-408-foundation

# 编译单个文件
g++ -std=c++17 "数据结构/01数组.cpp" -o array
./array

# 或者用 CMake（后续添加）
```

### 推荐学习顺序

```
入门 →  数组 / 链表 / 栈 / 队列
基础 →  二叉树 / 哈希表 / 二分查找
进阶 →  排序算法全系列
专题 →  逆波兰表达式 / 大数据查重 / 位图
```

每个 `.cpp` 文件都是独立可编译的，从 `main()` 函数开始阅读即可。
